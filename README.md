# Pattern & Number Analyzer

This Python project is designed to demonstrate the use of basic programming concepts in a simple way.

The project includes two main functionalities:

## 1. Star Pattern Generator

This module generates a right-angled triangle star pattern according to the number of rows entered by the user.
It uses nested loops for pattern creation.

## 2. Number Analyzer

This module checks whether the numbers in a given range are odd or even using conditional statements.

## Concepts Used

* Variables and Input/Output
* Functions
* for loop
* while loop
* Nested loops
* if-else statements
* range() function
* break statement

## Objective of the Project

The main objective of this project is to improve understanding of loops, conditions, and program flow in Python through practical implementation.
